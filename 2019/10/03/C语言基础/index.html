##1.结构体定义方式

结构体是【 struct name {成员变量列表}; 】这种结构的就是结构体。

一般用于搭配的类型定义 【 typedef A B ; 】意思：将 A 这种东西定义为一个类型，类型名称叫做 B 。

可以结合出以下合法形式：
1）typedef struct {成员变量} StructName;
☆ 最常用的方式
最常用的方法，将一个结构体生命为一个StructName类型，后续使用只需要将这整体看作一个类型即可。
使用：StructName data；
2）typedef struct strName {成员变量} StructName;
将结构体name定义为一个数据类型StructName，后续使用方法和上面相同
使用：StructName data；
可用但不推荐：struct strName data；
3）typedef struct strName {成员变量};
没啥卵用，相当于定义为类型
使用：struct strName data；
4）struct strName {成员变量}data1，data2;
定义一个结构体strName，并可以直接定义两个该结构体类型的结构体变量。

##3.理解复杂声明可用的“右左法则”：
从变量名看起，先往右，再往左，碰到一个圆括号就调转阅读的方向；括号内分析完就跳出括号，还是按先右后左的顺序，如此循环，直到整个声明分析完。举例：
1)int (func)(int p);
首先找到变量名func，外面有一对圆括号，而且左边是一个号，这说明func是一个指针；然后跳出这个圆括号，先看右边，又遇到圆括号，这说明 (func)是一个函数，
所以func是一个指向这类函数的指针，即函数指针，这类函数具有int类型的形参，返回值类型是int。
2)int (func[5])(int );
func 右边是一个[]运算符，说明func是具有5个元素的数组；func的左边有一个，说明func的元素是指针（注意这里的不是修饰func，而是修饰 func[5]的，原因是
[]运算符优先级比高，func先跟[]结合）。跳出这个括号，看右边，又遇到圆括号，说明func数组的元素是函数类型的指 针，它指向的函数具有int*类型的形参，返回
值类型为int类型。

##5.数组的两种表示
在定义一个长度为MAXSIZE的数组时，可以是 int a[MAXSIZE]; 或 int b;b = (int )malloc(sizeof(int)MAXSIZE);
所以说a虽然表示一个数组，但其本质是一个指向一个长度为MAXSIZE的数组首地址的 ☆指针☆ ，记住是指针
这也就有了，在传递数组为形参时，对应函数可以写: void func(int list) 或 void func(int a[])。

##6.指针的释放
在使用完指针后，若之后不再需要则需手动释放，现将该指针值为NULL，在free掉，因为系统不会自动回收。
误区：可能会认为将指针释放后当前指针所指向的数据也会被释放，但其实不会，指针变量是一个地址，它存储的空间对应的内存位置才是真的数据项，你只是释放掉了一个能寻址到
这个空间的方式，并不会对这个空间有任何影响，前提是你还有其他手段能寻址到这个空间，否则这个空间就会丢失。
