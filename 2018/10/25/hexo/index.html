hexo+github搭建以及迁移

Veröffentlicht am 2018-10-25

以防后面再望，mark一下

搭建github + hexo

安装所需工具

1.nodejs

首先终端运行 node -v，查看是否存在，若不存在，直接官网下载 recommend for most users 版本就行。（这样能省的安装homebrew，省事- -）

2. github

目前来说，使用Xcode会自动给你安github，电脑上肯定不会缺Xcode的，就懒得去查了

3.hexo

主角来了，安装hexo使用npm就行了，终端敲击：sudo npm -g hexo，等待一会就能下完，使用hexo -v看一下下

搭建

本地相关

1. 找个位置建个文件夹，叫hexo，blog都行。终端cd进来，运行 hexo init，初始化hexo相关，
2. 打开_config.yml,可以手动更改theme，忘了咋改了，反正也不用改就不找了
3. 使用hexo generate生成静态html文件，

github端

1. 终端运行cd ~/.ssh，查看是否存在ssh文件，没有的话就是not found directory
2. 生成ssh 终端运行 sudo ssh-keygen -t rsa -C "github-email@email.com"，接下来有三步，先输入id_rsa，然后回车，然后回车，结束。
3.打开你的电脑，搜索id_rsa.pub这个文件，使用sublime之类的软件打开，全部复制下来。
4.登录github网页，进入setting-> ssh&gub key -> add ssh key
5.随便起个title，把复制的从ssh开始的那个复制，保存。
6.终端输入 git config --global user.name "name"
          git config --global user.email "youremail"
7.ssh git@github.com 测试链接
8.将_config.yml中最后deploy下的网址改成git上建的新仓库 type改为git 分支是master就行了

结尾

此时使用sudo hexo deploy 同步新网页到github上

迁移hexo

1. 前面的过程到hexo init处都一样
2. 将原hexo目录全部复制粘贴进来，全部替换，省事
3.重新配置github端，将新sshx替换掉以前的，或者新加一个都行
